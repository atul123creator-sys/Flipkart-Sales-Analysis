# Flipkart-Sales-Analysis
📊 Flipkart Product Analysis
Project Overview

This project is an end-to-end Data Analytics solution built using Excel, SQL, and Power BI. The objective is to analyze Flipkart product data, uncover business insights, and build an interactive dashboard to support data-driven decision-making.

The dataset contains information about products across multiple categories, including selling price, MRP, discounts, ratings, number of reviews, and brands. Using SQL, I solved business-oriented queries to analyze pricing, premium products, brand performance, and category trends. The cleaned dataset was then visualized in Power BI using interactive KPIs, charts, and slicers to make the insights easy to explore.
